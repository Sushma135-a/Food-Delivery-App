FoodDeliveryApp (Core Java - Extended)
-------------------------------------
How to import into Eclipse:
1. Download and unzip this project.
2. In Eclipse -> File -> Import -> Existing Projects into Workspace -> Select root folder (the unzipped folder).
3. Run com.foodapp.Main as Java Application.

Data files are stored in the 'data' folder (created at runtime):
- users.txt
- orders.txt

Notes:
- This is a simple, console-based core-Java app (no DB, passwords stored in plaintext for demo only).
- Tested with Java 8+ and Eclipse.
